## Funktioner
- Välj om lösenordet ska innehålla stora bokstäver, små bokstäver och specialtecken.
- Generera lösenord med ett klick.
- Popup-ruta visar det genererade lösenordet.

## Tekniker
- Java
- Swing (GUI)
- Randomisering och stränghantering

## Hur man kör
1. Klicka på `Project1.java` för att se koden.
2. Kompilera och kör i valfritt Java-IDE.
